## ğŸ“Œ About the Project

This is a MERN stack (MongoDB, Express, React, Node.js) Feedback Management System.  
It supports **two types of users**:

- **Users** can:
  - Submit feedback
  - View their own submitted feedback

- **Admins** can:
  - View feedback from all users
  - Delete any feedback
